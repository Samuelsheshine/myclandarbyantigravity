# My Calendar (Antigravity) - 個人化行事曆與規劃工具

這是一個基於 React 18 與 Vite 開發的現代化個人行事曆應用程式。它不僅提供美觀的 UI 介面，還結合了行程管理與外部行事曆匯入功能。

## ✨ 功能特色

- **多維度行事曆**：直覺的月份與週視圖切換，方便管理長期與短期計畫。
- **每日行程規劃 (Daily Section)**：支援精細的時間段安排，讓你掌握每一天的進度。
- **目標追蹤 (Goals Tracking)**：專屬的目標設定區塊，協助你達成重要的 Milestone。
- **ICS 檔案匯入**：內建 ICS 解析器，可快速匯入現有的行事曆資料。
- **多語系支援**：內建國際化語系切換功能，支援繁體中文與英文。
- **現代化設計**：使用 Tailwind CSS 打造，並結合 Framer Motion 提供流暢的動畫體驗。

## 🛠️ 技術架構

- **核心框架**: React (Vite)
- **樣式設計**: Tailwind CSS
- **圖標庫**: Lucide React
- **動畫效果**: Framer Motion
- **語系管理**: 自定義翻譯模組

## 🚀 快速上手

### 1. 安裝依賴套件
```bash
npm install
```

### 2. 啟動開發伺服器
```bash
npm run dev
```

### 3. 打包與部署
```bash
npm run build
```

## 🎮 使用說明
- **新增行程**：點擊日曆或行程表中的時段即可加入新項目。
- **匯入行事曆**：使用導覽列中的匯入功能，上傳 `.ics` 格式的檔案。
- **切換語系**：可在設定或頁首處快速切換中文/英文。
